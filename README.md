# ig.news
 
O ig.news é uma aplicação criada para servir um Blog.
 
 
## Tecnologias 
 
Aqui estão as tecnologias utilizadas nesse projeto.
 
* Reactjs
* Nextjs
* Sass (CSS Preprocessor)
* Stripe (Meio de pagamento)
* FaunaDB (Banco de Dados)
* Next-Auth (Autenticação)
